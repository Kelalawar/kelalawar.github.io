JavaScript while Loop

While loop mengeksekusi perintah selama kondisi yang diberikan benar.

Code:

while(kondisi) {
// pernyataan atau perintah
}
  • condition - Ekspresi dievaluasi, jika mengevaluasi benar, blok kode dijalankan. Jika Salah, Loop akan berhenti.
  • statements Blok kode yang akan dieksekusi selama kondisi ini benar.

Ini adalah contoh sederhana.

var i = 0;
while(i < 20) {
 document.write('<div style="width: 50px; height: 50px; background: pink; margin: 5px; display: inline-flex; align-items: center;justify-content: center;">'+ i +'</div>');
 i++;
}

while(i < 20) angka 20 disini berfungi untuk menghitung berapa kali script harus mengulang perintah.
