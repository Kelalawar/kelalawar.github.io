Apa itu npm

npm adalah manajer package default untuk lingkungan runtime JavaScript Node.js. Ini terdiri dari command line client, juga disebut npm, dan database online package publik dan pribadi berbayar, yang disebut registri npm. Registri diakses melalui klien, dan package yang tersedia dapat diakses dan dicari melalui situs web npm. Manajer package dan registri dikelola oleh npm, Inc.

Pendapat saya

npm ini sangat penting, kenapa saya sebut npm ini sangat penting? karena banyak sekali package yang tidak tersedia di PKG tapi tersedia di npm , contoh nya package VScode tidak ada di PKG tapi ada di NPM walau pun namanya berbeda. menurut data terbaru NPM dipakai lebih dari 11 juta pengguna di seluruh dunia.
