CSS Colors

CSS color(warna) umumnya ditentukan dengan berikut ini:

  1. Color Name
  2. Hexadecimal Color Value
  3. RGB Color Value

Color Name Ini mendefinisikan warna yang valid dengan nama warnanya misalnya white (putih), black (hitam).

Contoh:

.css-warna {
   color: pink;
}

Hexadecimal Color Value

Hexadecimal juga dikenal sebagai nilai warna hex. Ini mendefinisikan warna yang valid dengan hash diikuti oleh huruf dan / atau angka misalnya. #ffc0cbff(pink), #ff0000(merah).

Contoh:

.css-warna {  
   color: #ffc0cbff;  
}

RGB Color Value

RGB adalah merah, hijau dan biru. Misalnya. rgb(255,192,203) (pink), rgb(255,0,0) (merah),

Angka yang dipisahkan dengan koma(,) mewakili warna merah, hijau dan biru.

Contoh:

color: rgb(255, 255, 255);
