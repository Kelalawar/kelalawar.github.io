lebih bagus vscode atau vim

Saya sangat bimbang untuk bisa menyimpulkan mana yang lebih bagus antara vscode ddengan vum karena dua-dua nya menurut saya adalah text editor tebaik, tapi berhubung judul artikel ini adalah menbabndingkan antara vscode dan vim jadi akan saya bandingkan menurut pengalaman saya mengunakan dua(2) text editor tersebut.

vscode

Visual Studio Code adalah editor source-code yang dibuat oleh Microsoft untuk Windows, Linux, dan macOS. Fitur termasuk dukungan untuk debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, dan embedded Git.

kelebihan vscode

  • mamiliki tampilan yang bagus dan moderen
  • bisa drag and drop serta klak klik klak klik mouse (maaf bahsa yang aneh ;D)
  • web server , decompiler dll sudah ada di vscode , bisa dibilang (all in one)
  • geratis kalian hanya tinggal kunjungi official website vscode dan langsung kallian pasang.
  • banyaksekali plugin yang bosa langsung di instal geratis.
  • memiliki banyak theme

kekeurangan vscode

sekalilagi ini menurut pengalaman saya.

  • LAG (f) “Saya tau ini bukan sesalahan softwhere , ini sepenuhnya kesalahan perngkat saya yang sangat sampah. "

vim

Vim; adalah perangkat lunak penyunting teks source-code untuk sistem operasi Unix. Penyunting ini adalah tiruan yang ditingkatkan dari penyunting teks vi buatan Bill Joy. Penulis Vim, Bram Moolenaar, membuat Vim berdasarkan port editor Stevie untuk Amiga. Vim mulai merilis versi ke publik pada tahun 1991.

kelebihan vim

  • ringan
  • banyaksekali plugin yang bisa dipasang
  • shortcut yang bisa kita atur sendiri
  • memiliki banyak theme
  • kalau memakai vim ditempat umum terlihat ganteng
  • geratis kalian hanya tinggal sudo apt-get vim maka akan langsung terpasang dii perangkat anda.
  • bisadipakai full keyboard only tanpa harus klak klik klak klik mouse

kekeurangan vscod

  • kalau belum terbiasa akan sulit sekali
  • webserver dan decompiler terpisah

kesimpulan vscode vs vim

menurut saya vim sangat powerfull dikarenakn selain ringan , hampir full bisa di customisasi dan vim juga tersedia di berbagai perangkat, pengunaannya pun lebih mudah kalau sudah terbiasa, bukan berarti vscode jelek ,vscode juga sangat bagus apalagi untuk orang-orang yang meyukai sesuatu yang simple,
