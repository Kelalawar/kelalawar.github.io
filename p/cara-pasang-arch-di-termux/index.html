apa itu Arch Linux

Menurut Wikipedia Arch Linux (dikenal juga dengan ArchLinux atau Arch) adalah sebuah distribusi Linux untuk komputer x86_64 (64 bit) yang didesain untuk menjadi ringan dan sederhana.

Pendekatan desain pengembang distro ini berfokus pada kesederhanaan, kebenaran program dan minimalisme. “Kesederhanaan” didefinisikan sebagai “tak perlu tambahan, modifikasi, atau komplikasi” sesuai sudut pandang pengembangnya, bukan sudut pandang pengguna.

Arch Linux adalah sebuah distribusi Linux berbasis biner, yang artinya, paket-paket aplikasi didistribusikan dalam bentuk telah terkompilasi. Berbeda dengan Gentoo yang mendistribusikan paket-paketnya dalam bentuk kode sumber.

pendapat saya tentang Arch

Menurut saya Arch Linux sangat bagus, apa lagi Arch Linux mempunyai banyak sekali tool, dan pemasangan nya pun sangat mudah , ukuran file yang akan di download sebesar 590 MB dan setelah proses instalasi Arch ukuran data termux akan menjadi 1,10 GB .

cara install

ada 3 cara Anda dapat memilih cara mana yang Anda suka.

sebelumnya anda harus menyiapkan storage :



termux-setup-storage

cara pertama

Metode ini dapat dijalankan dengan mengetuk tautan ini yang harus diunduh setupTermuxArch.sh ke ~/storage/downloads/direktori Anda di ponsel cerdas dan tablet Anda. Jika mengetuk pendek tautan ini tidak mengunduh setupTermuxArch.sh dari browser web Anda, coba ketuk lama dan pilih unduh file dari menu popup. Instal Arch Linux di Termux PROot hanya dengan menjalankan baris kode berikutnya di Termux:



bash ~/storage/downloads/setupTermuxArch.sh

cara kedua

Jalankan baris perintah berikut di bash untuk menginstal Arch Linux di Termux pada perangkat Anda. Jalankan setiap baris perintah berikut secara terpisah. Menjalankan semuanya sekaligus dapat menghasilkan kesalahan; Petunjuk, salin dan tempel berikut ini ke jendela Termux:


#install git
pkg install git

#download script
cd && git clone https://github.com/sdrausty/TermuxArch


#install script
bash TermuxArch/setupTermuxArch.sh

cara ketiga

Salin dan tempel baris perintah berikut ke shell perintah bash Termux Anda:


#install wget
pkg install wget

#download script
wget https://sdrausty.github.io/TermuxArch/setupTermuxArch.sh


#install script
bash setupTermuxArch.sh

perintah setelah penginstalan



pacman -S neofetch
pacman -S fish


Nano .bashrc

Tempel kan script ini di line paling bawah.



if [ -f /usr/bin/neofetch ]
then
neofetch
fish
fi
