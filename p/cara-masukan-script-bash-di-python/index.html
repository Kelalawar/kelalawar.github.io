Cara masukan script bash di python

Oct 30, 2021
2 minute read

Script bash Dan python tentunya berbeda, script bash tidak bisa begitusaja dimasukan didalam script python berlaku juga sebaliknya.

‚ùì: Apakah bash itu berguna didalam python ?

‚ùó‚ùó: tentusaja!!

Kalau Masih bingung aku akan memberikan contoh sederhana dengan code clear, ketika kamu membuat script bash kamu bisa langsung mengunakan code clear ini didalam script.

Contoh:


#! /bin/bash

clear # <== ini contoh nya
echo "Hello world!"
echo "Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua."

maka output nya akan seperti ini.

Code clear ini tidak bisa did sisipkan di script python mentah mentah seperti ini.


clear #<== ini code nya
print ("Hello World!")
print ("Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.")

Output nya akan seperti ini.

‚ùó‚ùóTerjadi eror!

‚ùì: Terus bagaimana cara mengatasinya?

Untuk memasukan bash ke python kamu harus menyisipkan code ini terlebih dahulu.


import os
def sh(script):
    os.system("bash -c '%s'" % script)

Contoh:



import os
def sh(script):
os.system("bash -c '%s'" % script)


sh("clear") #<== ini code nya
print ("Hello World!")
print ("Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.")

Dan hasilnya akan seperti ini.
