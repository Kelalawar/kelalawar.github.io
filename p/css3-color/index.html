RGBA Color Value

Tetapi dengan nilai alfa. rgba(merah, hijau, biru, alfa). Alpha adalah angka dalam kisaran 0,0 (sepenuhnya transparan) ke 1.0 (sepenuhnya buram) atau dalam istilah paling sederhana itu adalah nilai opacity.

Contoh:

color: rgba(255, 192, 203, 0.0);

color: rgba(255, 192, 203, 0.2);

 color: rgba(255, 192, 203, 0.4);

 color: rgba(255, 192, 203, 0.6);

 color: rgba(255, 192, 203, 0.8);

 color: rgba(255, 192, 203, 1.0);

HSLA Color Value

Nilai warna HSLA hanyalah perpanjangan dari nilai warna HSL.

Hampir sama halnya dengan rgba, hsla juga mempunyai nilai alfa. hsla(H, S, L, Alpha). Alpha adalah angka dalam kisaran 0,0 (sepenuhnya transparan) ke 1.0 (sepenuhnya buram).

Contoh:

   color: hsla(0, 100%, 50%, 0.0);

   color: hsla(0, 100%, 50%, 0.2);

   color: hsla(0, 100%, 50%, 0.4);

   color: hsla(0, 100%, 50%, 0.6);

   color: hsla(0, 100%, 50%, 0.8);

   color: hsla(0, 100%, 50%, 1.0);
