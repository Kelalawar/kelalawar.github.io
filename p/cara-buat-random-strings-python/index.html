Cara buat random strings python

Oct 20, 2021
1 minute read

Sekarang kita akan belajar membuat random strings dengan python,.

Pertama buatlah file ber extensi .py contoh aku akan membuat dengan nama ran.py

Contoh gambar:

dan pastikan kamu sudah menginstal python1/2/3 kalau belum bisa ikuti cara berikut.


apt update && apt upgrade

Contoh gambar:

setelah membuat file nya, lanjutkan untuk membuka file itu di text editor, saya mengunakan text editor micro.

Ini adalah script python random strings


import string    
import random 
import os
def sh(script):
    os.system("bash -c '%s'" % script)
S = 48  
sh("clear")
def main():
    ran = ''.join(random.choices(string.ascii_uppercase + string.digits, k = S))    
    print("The randomly generated string is : " + str(ran)) 
main()

import string    
import random 
import os
def sh(script):
    os.system("bash -c '%s'" % script)
S = 48  
sh("clear")
def main():
    ran = ''.join(random.choices(string.ascii_uppercase + string.digits, k = S))    
    print("The randomly generated string is : " + str(ran)) 
    yy = input("lagi? (y)(n) :")
    if yy == "y":
        main()
    else:
        print ("oke!")
main()
